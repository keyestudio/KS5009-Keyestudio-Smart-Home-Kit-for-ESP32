### 5.4.15 Project 8.1 Display Characters


#### **1. Description**

ご存知の通り、画面は人が電子機器とやり取りするための最良の方法の一つです。


#### **2. Component Knowledge**

1602は1行に16文字を表示できるモジュールです。2行あり、IIC通信プロトコルを使用します。

![画像50](../media/066e093f1711ada67d3309ddc9bdc66e.png)


#### **3. Control Pins**

| SDA | SDA |
| --- | --- |
| SCL | SCL |


#### **4. Test Code**

```c
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C mylcd(0x27,16,2);

void setup(){
  mylcd.init();
  mylcd.backlight();
}

void loop(){
  mylcd.setCursor(0, 0);
  mylcd.print("hello");
  mylcd.setCursor(0, 1);
  mylcd.print("keyestudio");
  //mylcd.clear();
}
```

#### **5. Test Result**

LCD1602の1行目に hello、2行目に keyestudio が表示されます。
