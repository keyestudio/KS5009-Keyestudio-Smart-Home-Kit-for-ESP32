### 5.4.15 Projet 8.1 Affichage de caractères


#### **1. Description**

Comme nous le savons tous, l'écran est l'un des meilleurs moyens pour les personnes d'interagir avec les appareils électroniques.


#### **2. Informations sur le composant**

1602 est un afficheur pouvant afficher 16 caractères par ligne. Il y a deux lignes, qui utilisent le protocole de communication IIC.

![image50](../media/066e093f1711ada67d3309ddc9bdc66e.png)


#### **3. Broches de contrôle**

| SDA | SDA |
| --- | --- |
| SCL | SCL |


#### **4. Code de test**

```c
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C mylcd(0x27,16,2);

void setup(){
  mylcd.init();
  mylcd.backlight();
}

void loop(){
  mylcd.setCursor(0, 0);
  mylcd.print("hello");
  mylcd.setCursor(0, 1);
  mylcd.print("keyestudio");
  //mylcd.clear();
}
```

#### **5. Résultat du test**

La première ligne du LCD1602 affiche hello et la deuxième ligne affiche keyestudio.
