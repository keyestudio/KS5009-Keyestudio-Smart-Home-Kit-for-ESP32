### 5.4.15 Project 8.1 Mostrar caracteres


#### **1. Descripción**

Como todos sabemos, la pantalla es una de las mejores formas para que las personas interactúen con los dispositivos electrónicos.


#### **2. Conocimiento del componente**

1602 es una línea que puede mostrar 16 caracteres. Hay dos líneas, que usan el protocolo de comunicación IIC.

![imagen50](../media/066e093f1711ada67d3309ddc9bdc66e.png)


#### **3. Pines de control**

| SDA | SDA |
| --- | --- |
| SCL | SCL |


#### **4. Código de prueba**

```c
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C mylcd(0x27,16,2);

void setup(){
  mylcd.init();
  mylcd.backlight();
}

void loop(){
  mylcd.setCursor(0, 0);
  mylcd.print("hello");
  mylcd.setCursor(0, 1);
  mylcd.print("keyestudio");
  //mylcd.clear();
}
```

#### **5. Resultado de la prueba**

La primera línea del LCD1602 muestra hello y la segunda línea muestra keyestudio.
