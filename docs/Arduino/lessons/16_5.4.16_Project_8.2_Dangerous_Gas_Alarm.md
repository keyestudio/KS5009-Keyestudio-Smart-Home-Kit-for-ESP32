### 5.4.16 Projet 8.2 Alarme de gaz dangereux


#### **1. Description**

Lorsqu'un capteur de gaz détecte une concentration élevée de gaz dangereux, le buzzer déclenchera une alarme et l'affichage indiquera "dangerous".


#### **2. Connaissances sur les composants**

**MQ2 Smoke Sensor** : C'est un dispositif de surveillance des fuites de gaz pour les habitations et les usines, adapté à la détection de gaz liquéfiés, benzène, alkyle, alcool, hydrogène ainsi que de la fumée. Notre capteur est câblé sur la broche numérique D et la broche de sortie analogique A, et il est connecté à D en tant que capteur numérique dans ce projet.

#### **3. Code de test**

```c
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

// Initialize LCD with I2C address 0x27, 16 columns and 2 rows
LiquidCrystal_I2C mylcd(0x27, 16, 2);

#define gasPin 23    // Gas sensor input pin
#define buzPin 25    // Buzzer output pin

// State flags for LCD display updates
boolean dangerDisplayed = 1;
boolean safetyDisplayed = 1;

void setup() {
  Serial.begin(9600);

  // Initialize LCD
  mylcd.init();
  mylcd.backlight();

  // Set pin modes
  pinMode(buzPin, OUTPUT);
  pinMode(gasPin, INPUT);

  // Display initial message
  mylcd.setCursor(0, 0);
  mylcd.print("safety");
}

void loop() {
  boolean gasVal = digitalRead(gasPin);  // Read gas sensor value
  Serial.println(gasVal);

  if(gasVal == 0)  // If dangerous gas detected
  {
    while(dangerDisplayed == 1)  // Update display if needed
    {
      mylcd.clear();
      mylcd.setCursor(0, 0);
      mylcd.print("dangerous");
      dangerDisplayed = 0;
      safetyDisplayed = 1;
    }

    // Sound alarm buzzer (short pulses)
    digitalWrite(buzPin, HIGH);
    delay(1);
    digitalWrite(buzPin, LOW);
    delay(1);
  }
  else  // No dangerous gas detected
  {
    digitalWrite(buzPin, LOW);  // Ensure buzzer is off

    while(safetyDisplayed == 1)  // Update display if needed
    {
      mylcd.clear();
      mylcd.setCursor(0, 0);
      mylcd.print("safety");
      dangerDisplayed = 1;
      safetyDisplayed = 0;
    }
  }
}
```

#### **4. Résultat du test**

L'écran affiche "safety" en état normal. Cependant, lorsque le capteur de gaz détecte certains gaz dangereux, tels que le monoxyde de carbone, à une certaine concentration, le buzzer émettra une alarme et l'écran affichera "dangerous".
