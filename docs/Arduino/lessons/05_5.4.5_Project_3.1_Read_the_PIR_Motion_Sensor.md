### 5.4.5 Project 3.1 Read the PIR Motion Sensor

![sensor de movimiento PIR](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. Descripción**

El sensor de movimiento PIR tiene muchos escenarios de aplicación en la vida diaria, como la lámpara de inducción automática de las escaleras, el grifo de inducción automática del lavabo, etc.

También es un sensor digital como los botones, que tiene dos estados con valores 0 o 1. Y se activará cuando las personas se estén moviendo.

Imprimiremos el valor del sensor de movimiento PIR a través del monitor serial.


#### **2. Pin de Control**

| sensor de movimiento PIR | 14 |
| --- | --- |
| \ |   |


#### **3. Código de prueba**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. Resultado de la prueba**

Cuando te quedas quieto frente al sensor, el valor leído es 0; muévete un poco y cambiará a 1.
