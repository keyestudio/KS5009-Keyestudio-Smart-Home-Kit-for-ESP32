### 5.4.5 Progetto 3.1 Leggere il sensore di movimento PIR

![immagine41](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. Descrizione**

Il sensore di movimento PIR ha molti scenari d'applicazione nella vita quotidiana, come la lampada automatica per le scale, il rubinetto automatico del lavabo, ecc.

È anche un sensore digitale come i pulsanti, che presenta due stati: 0 o 1. Rileva il movimento delle persone.

Stamparemo il valore del sensore di movimento PIR sul monitor seriale.


#### **2. Pin di controllo**

| Sensore di movimento PIR | 14 |
| --- | --- |
| \ |   |


#### **3. Codice di test**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. Risultato del test**

Quando stai fermo davanti al sensore, il valore letto è 0; se ti muovi un po', cambierà a 1.
