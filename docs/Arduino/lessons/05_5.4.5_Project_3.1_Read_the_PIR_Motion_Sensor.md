### 5.4.5 Projet 3.1 Lire le capteur de mouvement PIR

![image41](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. Description**

Le capteur de mouvement PIR a de nombreuses applications dans la vie quotidienne, comme les lampes à détection automatique pour escaliers, les robinets à détection automatique de lavabo, etc.

C'est également un capteur numérique, comme les boutons, qui a deux états (0 ou 1). Il détecte lorsqu'une personne se déplace.

Nous afficherons la valeur du capteur de mouvement PIR via le moniteur série.


#### **2. Broche de contrôle**

| Capteur de mouvement PIR | 14 |
| --- | --- |
| \ |   |


#### **3. Test Code**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. Résultat du test**

Lorsque vous restez immobile devant le capteur, la valeur lue est 0 ; en bougeant légèrement, elle passe à 1.
