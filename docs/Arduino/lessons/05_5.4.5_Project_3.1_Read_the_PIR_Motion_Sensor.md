### 5.4.5 Projekt 3.1 PIR Motion Sensor auslesen

![Bild41](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. Beschreibung**

Der PIR motion sensor hat viele Anwendungsszenarien im Alltag, wie zum Beispiel automatische Treppenbeleuchtung, automatische Sensor-Armatur am Waschbecken usw.

Er ist ebenfalls ein digitaler Sensor wie Taster und hat zwei Zustände 0 oder 1. Er reagiert, sobald sich Personen bewegen.

Wir geben den Wert des PIR motion sensor über den serial monitor aus.


#### **2. Steuerpin**

| PIR motion sensor | 14 |
| --- | --- |
| \ |   |


#### **3. Testcode**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. Testergebnis**

Wenn Sie ruhig vor dem Sensor stehen, ist der Messwert 0; bewegen Sie sich leicht, ändert er sich auf 1.
