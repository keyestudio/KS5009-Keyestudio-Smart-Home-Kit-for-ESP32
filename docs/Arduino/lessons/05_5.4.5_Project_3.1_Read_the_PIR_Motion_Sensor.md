### 5.4.5 プロジェクト 3.1 PIRモーションセンサーの読み取り

![画像41](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. 説明**

PIRモーションセンサーは階段の自動点灯照明や洗面台の自動水栓など、日常生活でさまざまな応用シーンがあります。

ボタンのようなデジタルセンサーで、0または1の2つの状態を持ち、人が動くと感知されます。

シリアルモニタにPIRモーションセンサーの値を表示します。


#### **2. 制御ピン**

| PIRモーションセンサー | 14 |
| --- | --- |
| \ |   |


#### **3. テストコード**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. テスト結果**

センサーの前で静止していると読み取り値は0になり、少し動くと1に変化します。
