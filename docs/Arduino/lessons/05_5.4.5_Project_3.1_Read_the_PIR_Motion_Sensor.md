### 5.4.5 Project 3.1 Lees de PIR-bewegingssensor

![afbeelding41](../media/c1518252606b111bfa66878a2bfcc965.png)


#### **1. Beschrijving**

De PIR-bewegingssensor heeft veel toepassingsscenario's in het dagelijks leven, zoals automatische bewegingsgestuurde verlichting voor trappen, automatische bewegingsgestuurde kraan bij wastafels, enz.

Het is ook een digitale sensor zoals knoppen, die twee toestandswaarden heeft: 0 of 1. En hij geeft een signaal wanneer mensen bewegen.

We zullen de waarde van de PIR-bewegingssensor afdrukken via de seriële monitor.


#### **2. Besturingspin**

| PIR-bewegingssensor | 14 |
| --- | --- |
| \ |   |


#### **3. Testcode**

```c
#define pyroelectric 14

void setup() {
  Serial.begin(9600);
  pinMode(pyroelectric, INPUT);
}

void loop() {
  boolean pyroelectric_val = digitalRead(pyroelectric);
  Serial.print("pyroelectric value = ");
  Serial.println(pyroelectric_val);
  delay(200);
}
```

#### **4. Testresultaat**

Als je stil voor de sensor staat, is de uitlezing 0; beweeg je een beetje, dan verandert deze naar 1.
