### 5.4.1 Proyecto 1.1 Parpadeo de LED


#### **1 Descripción**

![image-20230927115910677](../media/image-20230927115910677.png)

Hemos instalado el controlador de la placa de desarrollo ESP32 PLUS.

En la primera lección, realizaremos un experimento para hacer parpadear el LED.

Conectemos GND y VCC para alimentar. El LED estará encendido cuando la terminal de señal S esté en nivel alto; por el contrario, el LED se apagará cuando la terminal de señal S esté en nivel bajo.

Además, se puede mostrar una frecuencia de parpadeo diferente ajustando el tiempo de retardo.


#### **2 Principio de funcionamiento**

El LED también es un diodo emisor de luz, que puede fabricarse como un módulo electrónico. Brillará si controlamos los pines para que salgan en nivel alto; de lo contrario estará apagado.


#### **3 Parámetros**

| Voltaje de trabajo | DC 3~5V |
| --- | --- |
| Corriente de trabajo | <20mA |
| Potencia | 0.1W |


#### **4 Pin de control**

| LED amarillo | 12 |
| --- | --- |
| \ |   |


#### **5 Código de prueba**

```c
#define led_y 12  //Define the yellow led pin to 12

void setup() {    //The code inside the setup function runs only once
  pinMode(led_y, OUTPUT);  //Set pin to output mode
}

void loop() {     //The code inside the loop function will always run in a loop
  digitalWrite(led_y, HIGH);  //Light up the LED
  delay(200);     //Delay statement, in ms
  digitalWrite(led_y, LOW);   //Close the LED
  delay(200);
}
```

#### **6. Resultado de la prueba**

Después de cargar el código, puede ver los LEDs blanco y amarillo parpadeando juntos.
