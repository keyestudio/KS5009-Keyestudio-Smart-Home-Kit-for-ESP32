### 5.4.9 Project 5.1 Deur bedienen


#### **1. Beschrijving**

Automatische deuren en ramen hebben een aandrijfmechanisme nodig, dat met een 180° servo en enkele sensoren automatischer wordt. Door een regendruppelsensor toe te voegen kun je bereiken dat ramen bij regen automatisch sluiten. Als je een RFID toevoegt, kunnen we het effect realiseren van swipen om de deur te openen, enzovoort.


#### **2. Componentkennis**

**Servo:** Een servo is een positioneringsapparaat dat bestaat uit een behuizing, een printplaat, een kernloze motor, een tandwiel en een positiedetector.

Het werkingsprincipe is dat de servo het signaal van de MCU of ontvanger ontvangt en een referentiesignaal met een periode van 20ms en een pulsbreedte van 1,5ms produceert, vervolgens de verkregen DC-biasspanning vergelijkt met de spanning van de potentiometer en het spanningsverschil berekent.

De IC op de printplaat bepaalt de draairichting en drijft daarna de kernloze motor aan om te starten. De kracht wordt via het reductietandwiel naar de hevel overgebracht, en het signaal wordt door de positiedetector teruggegeven om te beslissen of de positionering bereikt is. Dit is geschikt voor besturingssystemen die een constante hoekverandering vereisen en die behouden kunnen worden.

Wanneer de motorsnelheid constant is, wordt de potentiometer via de trapreductie aangedreven, waardoor het spanningsverschil 0 wordt en de motor stopt met draaien. Over het algemeen is het hoekbereik van servo-rotatie 0°–180°.

De pulsperiode van de stuursservo is 20ms, de pulsbreedte is 0,5ms ~ 2,5ms, en de overeenkomstige positie is -90° ~ +90°. Hier is een voorbeeld van een 180° servo:

![afbeelding van een 180° servo](../media/708316fde05c62113a3024e0efb0c237.jpeg)

In het algemeen heeft een servo drie draden in bruin, rood en oranje. De bruine draad is massa, de rode is de positieve voedingsdraad en de oranje is de signaaldraad.

#### **4. Testcode**

```c
#include <ESP32Servo.h>
Servo myservo;  // create servo object to control a servo
                // 16 servo objects can be created on the ESP32

int pos = 0;    // variable to store the servo position
// Recommended PWM GPIO pins on the ESP32 include 2,4,12-19,21-23,25-27,32-33
int servoPin = 13;

void setup() {
    // Allow allocation of all timers
    ESP32PWM::allocateTimer(0);
    ESP32PWM::allocateTimer(1);
    ESP32PWM::allocateTimer(2);
    ESP32PWM::allocateTimer(3);
    myservo.setPeriodHertz(50);    // standard 50 hz servo
    myservo.attach(servoPin, 1000, 2000); // attaches the servo on pin 18 to the servo object
    // using default min/max of 1000us and 2000us
    // different servos may require different min/max settings
    // for an accurate 0 to 180 sweep

}

void loop() {
    for (pos = 0; pos <= 180; pos += 1) { // goes from 0 degrees to 180 degrees
        // in steps of 1 degree
        myservo.write(pos);    // tell servo to go to position in variable 'pos'
        delay(15);             // waits 15ms for the servo to reach the position
    }
    for (pos = 180; pos >= 0; pos -= 1) { // goes from 180 degrees to 0 degrees
        myservo.write(pos);    // tell servo to go to position in variable 'pos'
        delay(15);             // waits 15ms for the servo to reach the position
    }

}
```
